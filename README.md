# 🚗 Parking Lot Management System (C)

This is a simple parking system made using the C language.
It runs in the console and helps manage parking spaces easily.

📌 What this program does

* Lets you **add a car** to a parking slot
* Lets you **remove a car** from a slot
* Helps you **search a car** using its registration number
* Shows how many parking spaces are **occupied and free**

---

## 🛠️ Language Used

* C programming language
* Libraries used:

  * stdio.h
  * stdbool.h
  * string.h
  * stdlib.h

---

## ⚙️ How it works

* The parking lot has **30 slots**

* Each slot stores:

  * Owner name
  * Vehicle registration number
  * Slot number

* The program uses a menu where you choose what you want to do

---

## ▶️ How to run the program

### Step 1: Compile

```bash
gcc parking.c -o parking
```

### Step 2: Run

```bash
./parking
```

---

## 📋 Menu options


1. Allocate Parking Space
2. Search for a car's parking slot
3. Parkinglot Status
4. Deallocate a parking slot
5. Exit

## ⚠️ Limitations

* Only **30 parking slots**
* Data is **not saved** after closing the program
* Owner name cannot have spaces
* Basic error handling
